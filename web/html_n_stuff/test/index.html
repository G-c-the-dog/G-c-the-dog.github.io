  • Discord

End of reordering

The remainer of the old page.

  • Uninstall Tool

    It's an incredibly useful application if you want to erase different apps pretty easily, and it's free*.

    * - the "better unistall" (forced unistall or smth) is still behind a paywall, but the base unistall is already enouth for anyting, really.

  • MSI afterburner

    It allows overclocking your graphics card and monitoring different stats.

  • Handbrake

    This app can convert videos from almost any format.

  • Visual studio community 2019

    This is a really useful programming IDE with tons of addons for different use cases.

  • Open Broadcast Software or OBS

    OBS can capture videos w/sound and livestream directly into Twich or Youtube.

    It's a litle bit confusing at first, but after some time and tutorials read you'll love it.

  • Intelligent standby list cleaner or ISLC

    It is designed to help you monitor and clear the memory standby list.

  • K-lite codec pack

    It's basic version contains almost all codecs needed for daily media consumtion, it even provides a very potent media player.
