The one and only - The /tp command.

It allows you to teleport youself or any entity to a certain location, facing a certain block.

The syntax is very straightforward:

/tp @s x y z facing x y z

You can click text for extra information about it's components.

>>/tp. The command itself

>>@s. Teleported entity, can be @e or @a... etc with specifications to tp a certian entity.

>>@e or @a... can be replaced by @e - every entity, @a - every player, @s - the command sender, @p - closest player to the command block/commnad sender

>>specifications. You can narrow the target list by using a filter, such as @e[type=minecraft:mob_name], @e[distance=x], @e[limit=x]

>>@e[type=minecraft:mob_name] The "mob_name" can be replaced by any mob's name, like "pig" or "cow", or "villager", or "creeper"... you get the point.

>>@e[distance=x]. Right now the command will tp any entity at distance "x" from the sender. If replaced with "..x", the command will tp any entity closer, than x. If replaced with "x..", any entity farther, than x. And, finally, if replaced with "x..y", the command will tp any entity farther, than x, but closer, than y, from the sender.

>>@e[limit=x]. The command will be used on x amount of entities. For example, if x set to 0, the comand will do literally nothing (will tp zero entities).

>>x y z. The coords it's being tp'd to

>>facing x y z. Makes the tp'd entity face the specified coords
