Страница с полезными приложениями для Windows на новой системе.

Сдесь я расскажу о некоторых приложениях, которые я считаю полезными.

  • Uninstall Tool

    Это очень полезное приложение для простого и быстрого удаления приложений, и оно бесплатное*.

    * - за "Принудительное удаление" всё ещё нужно платить, но базового удаления уже хватит для почти для всего.

  • MSI afterburner

    Он позволяет разгонять видеокарту и следить за различными параметрами.

  • Handbrake

    Данное приложение позволяет конвертировать видео практически из любого формата.

  • Visual studio community 2019

    Это очень полезная ИСР (IDE на англ.) с огромным количеством дополнений для различный ситуаций.

  • Open Broadcast Software или OBS

    OBS может сниамть видео со звуком и транслировать сразу на Twich или Youtube.

    В нём изначально можно запутаться, но со временем и после прочтения гайдов он вам понравится.

  • Intelligent standby list cleaner или ISLC

    Оно сделано для очищения кеша из оперативной памяти.

  • K-lite codec pack

    It's basic version contains almost all codecs needed for daily media consumtion, it even provides a very potent media player.

  • Steam

    Steam - это платформа, используемая для скачивания и обмена приложения и игр.

  • Discord

    Это приложение позволяет людям находить места, где они могут обсуждатьразличные темы.
